# sumintern

本仓库用于存放个人学习与实验项目。

## 项目列表

### snake-vim

一个基于 Vite 的网页版霓虹贪吃蛇游戏，支持 Vim 模式控制。

![snake-vim 预览](./image/snake-vim.png)

进入项目：[snake-vim](./snake-vim)

### post-display

一个基于 Vue 3 + Vite + ECharts 的纯前端数据大屏，展示帖子数量、标签分布与敏感舆论管控。

进入项目：[post-display](./post-display)

## 运行项目

```bash
cd snake-vim   # 或 cd post-display
npm install
npm run dev
```

## 说明

- 根目录下的 `.node/` 是本地 Node.js 运行环境，不参与 Git 提交。
- 各子项目依赖请进入对应目录后单独安装。
